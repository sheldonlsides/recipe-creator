# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct. To keep the
text current, we link to the canonical version rather than reproducing it here:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By contributing code, filing issues, or joining discussions, you agree to uphold it.

## Reporting

Report unacceptable behavior **privately** through GitHub's private reporting: go to the
repository's **Security** tab and click **Report a vulnerability**. Reports are reviewed promptly
and handled confidentially.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1.
